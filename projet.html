    VAE

    Description du projet :

    Pour ce projet, notre mission consistait à concevoir, en équipe de cinq personnes, une application de vente aux enchères similaire à eBay, dans le cadre d'une semaine de travail au sein d'un environnement d'apprentissage par projet (SAE).

    Objectifs :

    L'objectif de ce projet était de mettre en pratique toute les compétences apprise durant l'année et de les utilisée dans un travail de groupe, il avait également pour objectif de nous préparer au monde professionnelle en travaillant en équipe avec des postes pour chaque membre.

    Technologies utilisées :

    • Java FX
    • SQL
    • JDBC
    • CSS

    Résultats obtenus :

    Nous avons obtenus de très bon résultat avec une application fonctionnelle regrouppant les principales fonctionnalité requise a une appli de vente au enchères
